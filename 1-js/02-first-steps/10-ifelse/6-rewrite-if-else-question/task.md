# Rescrieți 'if..else' în '?'

Rescrieți `if..else` folosind mai mulți operatori ternary `'?'`.

Pentru lizibilitate, este recomandat să împărțiți codul în mai multe linii.

```js
let message;

if (login == 'Angajat') {
  message = 'Bună ziua!';
} else if (login == 'Director') {
  message = 'Salutări!';
} else if (login == '') {
  message = 'Nu te poți loga';
} else {
  message = '';
}
```
